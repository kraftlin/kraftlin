# Module kraftlin-config-core

Platform-agnostic type-safe configuration using Kotlin property delegation.

Provides [io.github.kraftlin.config.AbstractConfig] as the base for config classes and
[io.github.kraftlin.config.wrapConfig] for binding to YAML files.

# Package io.github.kraftlin.config

- [io.github.kraftlin.config.AbstractConfig]: Base class for type-safe config declarations.
- [io.github.kraftlin.config.wrapConfig]: Creates a config wrapper for a given file path.
