# Module kraftlin-message-bungee

BungeeCord-specific message sending extensions.

Provides `CommandSender.message` extensions that serialize Adventure components via
`BungeeComponentSerializer` for sending on BungeeCord proxies.

# Package io.github.kraftlin.message.bungee

- [io.github.kraftlin.message.bungee.message]: Extension functions on BungeeCord's `CommandSender`.
