# Module kraftlin-config-velocity

Velocity-specific config loading helpers.

Provides [io.github.kraftlin.config.velocity.wrapConfig] for binding to a Velocity plugin's data directory.

# Package io.github.kraftlin.config.velocity

- [io.github.kraftlin.config.velocity.wrapConfig]: Wraps a Velocity plugin's config file.
- [io.github.kraftlin.config.velocity.loadSqlConfiguration]: Loads database settings from `database.yml`.
