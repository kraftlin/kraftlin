# Module kraftlin-config-bungee

BungeeCord-specific config loading helpers.

Provides [io.github.kraftlin.config.bungee.wrapConfig] for binding to a BungeeCord plugin's data folder.

# Package io.github.kraftlin.config.bungee

- [io.github.kraftlin.config.bungee.wrapConfig]: Wraps a BungeeCord plugin's config file.
- [io.github.kraftlin.config.bungee.loadSqlConfiguration]: Loads database settings from `database.yml`.
